Use Cases

rx-platform is a software framework that can be used in any kind of projects involving industrial process. It enables Rapid Application Development of several types of products: MES Systems, Gateways, HMI, Simulation Tools, IoT and Industrial IoT Applications...

Deployment

Latest Version of rx-platform binaries and SDK-s are available for download or you can download source from respository.
Currentlly two platform/compiler combinations are fully supported:
- Windows 7 and later (MSVC)
- GNU Linux Kernels 4.4 (GCC)
